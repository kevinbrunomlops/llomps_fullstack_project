## Team Goals
- Build a working full-stack AI travel chatbot
- Deliver on time
- Ensure fair contribution from all members
- Learn and improve together

## Responsibilities
- Member 1: Dataset and travel content
- Member 2: Backend and AI integration
- Member 3: Frontend and UI
- But also overlapping cooperation when necessary

All members help with testing, documentation, and presentation.

## Communication
- Main channel: Discord
- Respond when available
- Inform team if delayed or blocked
- Respect each other

## Meetings
- 3 short sync meetings per week
- 1 planning meeting per week
- Extra meetings near deadlines

## Git Workflow
- No pushes directly to main
- Use personal feature branches
- Use pull requests
- Review before merge

## Task Management
- Use GitHub Issues
- Use Kanban board (Github Projects)
- Update progress regularly

## Code Quality
- Write clean and readable code
- Test before pushing
- Keep structure organized

## If Problems Arise
- Raise issues early
- Solve respectfully
- Support each other

## Presentation
- Everyone presents
- Everyone understands the project

## Agreement
We agree to work professionally, respectfully, and contribute fairly.
